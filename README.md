# Electron Notas Minimalistas

Aplicación de notas minimalista en Electron con enfoque en escritura, sonido de teclado agradable, panel lateral y modo Zen.

## ¿Para qué sirve?

- Escribir notas rápidas con formato básico
- Guardar y abrir archivos .txt en la carpeta de Documentos
- Enfocarte en el texto con un modo Zen

## Cómo ejecutar

1. Instala dependencias:

```
npm install
```

2. Ejecuta la app:

```
npm start
```

## Uso rápido

- Guardar: Ctrl+S
- Abrir: Ctrl+O
- Negrita: Ctrl+B
- Cursiva: Ctrl+I
- Subrayar: Ctrl+U
- Imprimir / PDF: Ctrl+P
- Panel lateral: Ctrl+\
- Modo Zen: Ctrl+Shift+Z

## Carpeta de guardado

Los archivos se guardan en:

```
Documentos/MinimalNotePad
```
